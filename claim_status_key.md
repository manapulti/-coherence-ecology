# Claim Status Key

| Status | Meaning |
|---|---|
| Solid | Supported by direct evidence or widely accepted external sources |
| Inferred | Reasonable synthesis from available evidence |
| Speculative | Possible but not established |
| Interpretive | Meaning-level reading, not empirical proof |
| Unsupported | Not currently supported by located evidence |
| Overclaim | Exceeds available evidence and should be corrected |
