# Layer Separation Map

## Five-layer discipline

| Layer | Question | Failure if collapsed |
|---|---|---|
| Reality | What happened / exists? | Myth replaces fact |
| Observation | What was measured or directly seen? | Feeling becomes proof |
| Model | What representation explains or organizes it? | Useful model becomes ontology |
| Interpretation | What does the model mean? | Meaning becomes evidence |
| Ontology | What is claimed to exist? | Speculation becomes certainty |

## Claim-status key

| Status | Meaning |
|---|---|
| Solid | Well-supported within current evidence |
| Inferred | Reasonable but indirect |
| Speculative | Possible but unsettled |
| Interpretive | Meaning-level, not empirical proof |
| Overclaim | Exceeds evidence and must be softened or corrected |

## Use against J-space overclaim

Observation: J-lens can read and manipulate some verbalizable internal representations.  
Model: These representations function like a global workspace.  
Interpretation: This resembles some functional features associated with cognitive access.  
Ontology: Consciousness / personhood / subjective experience remains unproven.
