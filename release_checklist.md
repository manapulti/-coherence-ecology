# Release Checklist

## Before first public release

- [ ] Review README for overclaims.
- [ ] Confirm author/steward line reads Anthony Barlow.
- [ ] Add license.
- [ ] Add CITATION.cff.
- [ ] Add exported evidence files or link to evidence archive.
- [ ] Generate SHA-256 hashes for all evidence files.
- [ ] Fill hash ledger.
- [ ] Confirm no private/sensitive material is included accidentally.
- [ ] Confirm collaborator attributions are correct.
- [ ] Mark AI assistance as drafting/retrieval/tool assistance only.
- [ ] Tag GitHub release `v0.1.0-provenance-scaffold`.

## Optional stronger preservation

- [ ] Create GitHub release ZIP.
- [ ] Archive release to Zenodo or OSF.
- [ ] Add DOI to README after archive.
- [ ] Preserve original file exports in read-only archive.
